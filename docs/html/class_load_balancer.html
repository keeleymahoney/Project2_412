LoadBalancer::LoadBalancer ( char type,
int num )

Constructs a LoadBalancer object.

Initializes the LoadBalancer with a specified job type ('s' or 'p') and a given number of servers. Each server is added to the LoadBalancer's server pool.

Parameters
typeThe job type for this LoadBalancer ('s' or 'p').
numThe initial number of servers for the LoadBalancer.

◆ assignRequest()

void LoadBalancer::assignRequest ( )

Assigns pending requests to available servers.

The function iterates through all servers, and if a server is not busy, it assigns the first request in the queue to that server.

◆ generateRequests()

void LoadBalancer::generateRequests ( )

Generates a random number of new requests and adds them to the queue.

A random number of requests (between 1 and 4) are generated and added to the request queue based on the job type ('s' or 'p').

◆ scale()

void LoadBalancer::scale ( )

Scales the number of servers based on the current queue size.

The function determines the number of servers needed based on the queue size (calculated as ceil(requests.size() / 20.0)), and scales the number of servers up or down accordingly.
