\doxysubsection{Detailed Description}
Entry point for the load balancing simulation. 

Simulates the behavior of load balancers managing requests across web servers. Users can specify the number of time cycles and servers for both streaming (\textquotesingle{}s\textquotesingle{}) and processing (\textquotesingle{}p\textquotesingle{}) job types. The program generates requests, processes them, and scales servers dynamically based on time cycles.
